通过东方财富网爬取上证综指股票信息，之后进行数据清洗和数据分析。
文件结构：
main_spider.py          主函数
get_all_url.py          获取0、3、6开头的股票代码
html_downloader.py      抓取股票的历史数据网页
html_parser.py          从抓取的股票网页中获得数据
html_output_csv.py      把数据存入CSV文件

V1.0 功能：从网页上抓取股票历史信息，并将每个股票存入股票代码命名的CSV文件
           判断CSV文件是否存在，存在则不继续抓取
           加入异常保护，即使一个网页抓取失败程序可继续执行

           不再对V1.0版本进行改进

           取消了原来的文件结构，将所有功能集中在new_spider中，不到100行
